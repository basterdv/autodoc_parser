<h3> Для запуска проекта нужно выполнить </h3>

<p> 1. Установить виртуальное окружение и активировать его </p>   

        python -m venv venv

        venv/Scripts/activate

<p> 2. Установить зависимости </p>

        pip install -r requirements.txt 

<p> 3. Запускаем </p>

        py main.py
